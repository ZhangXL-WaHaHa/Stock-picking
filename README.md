# 多策略股票筛选器

基于 FastAPI 的 A 股多策略自动选股工具，支持四种独立筛选策略自由切换，配合 Web 界面、定时任务、飞书/邮件推送、胜率追踪与失败分析。

> **风险提示**：股市有风险，投资需谨慎。本工具仅为量化筛选辅助，不构成任何投资建议。筛选结果基于历史数据和技术指标，无法预测未来走势，不保证盈利。使用者应结合自身风险承受能力独立做出投资决策，并自行承担全部投资风险。

## 筛选策略

### 一夜持股法

尾盘买入、次日开盘卖出，捕捉隔夜溢价。

| 序号 | 规则 | 条件 |
|------|------|------|
| 0 | 大盘环境 | 上证跌超 0.5% 时不出信号 |
| 1 | 涨幅区间 | 2% ~ 6% |
| 2 | 量比 | > 1 |
| 3 | 换手率 | 5% ~ 10% |
| 4 | 流通市值 | 50亿 ~ 300亿 |
| 5 | 收盘价位置 | 收盘价处于当日上半区（排除冲高回落） |
| 6 | 涨停+缩量回调 | 近 15 日有涨停且后续缩量，主力控盘信号 |
| 7 | VWAP/尾盘信号 | VWAP 偏离度 + 尾盘量能变化 |

### 均线回踩

均线多头排列、缩量回踩 MA10 后企稳反弹。

| 序号 | 规则 | 条件 |
|------|------|------|
| 0 | 大盘环境 | 上证跌超 0.5% 时不出信号 |
| 1 | 均线多头排列 | MA5 > MA10 > MA20 |
| 2 | 回踩 MA10 | 收盘价距 MA10 在 2% 以内 |
| 3 | 缩量 | 近 3 日均量 < 前 10 日均量的 70% |
| 4 | 企稳反弹 | 当日收盘 > 前日收盘 |

### RPS 强势突破

相对强度排名前 10%，横盘整理后突破新高。

| 序号 | 规则 | 条件 |
|------|------|------|
| 0 | 大盘环境 | 上证跌超 0.5% 时不出信号 |
| 1 | RPS >= 90 | 60 日涨幅排名全市场前 10% |
| 2 | 创 20 日新高 | 收盘价突破近 20 日最高价 |
| 3 | 10 日振幅 < 10% | 突破前处于横盘整理状态 |
| 4 | 流通市值 | 30亿 ~ 500亿 |

### 低位首板

低位首次涨停，等待缩量回调后介入。

| 序号 | 规则 | 条件 |
|------|------|------|
| 1 | 当日涨停 | 涨幅 >= 9.8% |
| 2 | 低位 | 距 60 日最高点跌幅 > 30% |
| 3 | 首次涨停 | 近 60 日内无其他涨停记录 |
| 4 | 流通市值 | 30亿 ~ 300亿 |

> 低位首板策略筛出的股票标注为"观察中"，不建议当天追入，等缩量回调 2-3 天再考虑介入。

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py
```

浏览器打开 http://localhost:8000

## 功能说明

### Web 界面

- **策略切换**：页面顶部四个策略标签自由切换，策略说明和筛选规则联动展示
- **立即筛选**：点击按钮手动触发当前策略的全市场扫描
- **定时筛选**：配置周一至周五的自动执行时间，支持多个时间点
- **胜率追踪**：自动记录 5 星股票的买卖盈亏，展示总胜率和累计收益
- **失败分析**：每笔亏损交易自动分析原因（大盘拖累、跳空低开、追高接盘、放量滞涨）
- **实时结果**：刷新页面自动加载最新筛选数据

### GitHub Actions 自动推送

通过 GitHub Actions 在交易日尾盘自动执行筛选，并将结果推送到飞书/邮箱。

**触发时间**：周一至周五 14:30、14:40、14:50（北京时间）

也支持在 GitHub Actions 页面手动触发（workflow_dispatch）。

## 推送配置

在 GitHub 仓库 **Settings > Secrets and variables > Actions** 中添加以下 Secrets。

飞书和邮箱**各自独立**，配了哪个就推哪个，都不配则只执行筛选不推送。

### 飞书机器人

| Secret | 说明 |
|--------|------|
| `FEISHU_WEBHOOK_URL` | 飞书自定义机器人的 Webhook 地址 |

获取方式：飞书群 > 设置 > 群机器人 > 添加自定义机器人 > 复制 Webhook 地址。

### 邮箱推送

| Secret | 说明 | 示例 |
|--------|------|------|
| `SMTP_HOST` | SMTP 服务器地址 | `smtp.qq.com` / `smtp.163.com` |
| `SMTP_PORT` | SMTP 端口（默认 465） | `465` |
| `SMTP_USER` | 发件邮箱 | `xxx@qq.com` |
| `SMTP_PASS` | 邮箱授权码（非登录密码） | QQ邮箱：设置 > 账户 > 生成授权码 |
| `EMAIL_TO` | 收件邮箱，多个用逗号分隔 | `a@xx.com,b@xx.com` |

## 项目结构

```
stock-screener/
├── main.py              # FastAPI 主服务，Web 界面 + 定时任务
├── screener.py          # 选股入口，根据策略分发
├── data_fetcher.py      # 行情数据获取（腾讯财经 + 东方财富）
├── strategies/
│   ├── __init__.py      # 策略注册表
│   ├── overnight.py     # 一夜持股法
│   ├── ma_pullback.py   # 均线回踩
│   ├── rps_breakout.py  # RPS 强势突破
│   └── low_first_zt.py  # 低位首板
├── trade_tracker.py     # 胜率追踪 + 失败分析
├── settle_trades.py     # 每日结算脚本
├── notify_feishu.py     # 飞书/邮件推送
├── models.py            # 数据模型定义
├── database.py          # SQLite 数据库操作
├── requirements.txt     # Python 依赖
├── templates/
│   └── index.html       # Web 页面模板
├── static/
│   └── style.css        # 样式文件
└── .github/
    └── workflows/
        └── stock-screen.yml  # GitHub Actions 工作流
```

## 数据来源

| 数据 | 来源 | 用途 |
|------|------|------|
| 实时行情 | 腾讯财经 | 价格、涨跌幅、量比、换手率、流通市值 |
| 历史 K 线 | 腾讯财经 | 均线计算、涨停判断、RPS 排名 |
| 分时数据 | 腾讯财经 | VWAP 偏离度、尾盘量能分析 |
| 题材概念 | 东方财富 | 股票所属概念板块标签 |

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | Web 主页 |
| GET | `/api/strategies` | 获取所有策略列表及规则说明 |
| GET | `/api/screen?strategy=overnight` | 执行指定策略筛选 |
| GET | `/api/latest?strategy=overnight` | 获取指定策略最近一次结果 |
| GET | `/api/winrate` | 获取胜率统计和交易记录 |
| GET | `/api/schedules` | 获取定时任务配置 |
| POST | `/api/schedules` | 更新定时任务配置 |
